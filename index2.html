StackOverflow data set analysis

Following visualization/analysis is done for the stackoverflow dataset for JAVA tags:

  1. Visualize top packages discussed and pie chart for topics based on text analyzis
  2. Bar graph Monthly analysis for each topic/package for number of Questions, Answers and Accepted Answers.
  3. Sunburst for Weekly analysis of each topic/package helpful to have insights if you should ask quesiton at beginning of week, mid of the week or towards the end of the week
  4. Time series and Network analysis for Mapping popular users to topics they discuss and have information about that user's activity.
  5. SVM classification model for classifying if number of posts on day of week are High or Low. While training number of posts above median are assumed as High, below median are assumed as Low.
